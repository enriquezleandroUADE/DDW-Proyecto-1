# Portfolio Personal

Bienvenido/a a mi portfolio personal. Este repositorio reúne una presentación breve sobre mí y mi perfil, organizada de forma simple para que puedas encontrar rápido lo más importante.

## Estructura

El portfolio se divide en **3 secciones principales**:

### 1) Index

La página principal del portfolio.

- Presentación general
- Resumen de habilidades/stack
- Acceso rápido al resto de secciones

### 2) Intereses

Una sección para mostrar lo que me motiva y en qué áreas me interesa crecer.

- Temas que estudio actualmente
- Áreas de interés (por ejemplo: desarrollo web, UX/UI, backend, etc.)
- Objetivos de aprendizaje

### 3) Contacto

Información para comunicarte conmigo.

- Links a redes/perfiles (GitHub, LinkedIn, etc.)
- Email u otros medios
- Disponibilidad (opcional)

## Cómo navegarlo

- Empezá por **Index** para una vista general.
- Pasá a **Intereses** para conocer mis focos actuales.
- Usá **Contacto** si querés hablar conmigo o colaborar.
